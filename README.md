# Customizable Dark Theme Chrome Extension

## Introduction

This Chrome extension allows users to convert any webpage to a customizable dark theme. Users can adjust the darkness level and customize various aspects of the theme, such as color schemes, font size, and background patterns.

## Features

- Adjustable darkness level slider
- Customizable color schemes
- Font size adjustment
- Background pattern selection
- Preview section to see changes before applying them
- Saves and retrieves customization settings using `chrome.storage`

## Installation

1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on the "Load unpacked" button and select the directory where you cloned or downloaded this repository.
5. The extension should now be installed and visible in the extensions toolbar.

## Usage

1. Click on the extension icon in the Chrome toolbar to open the popup.
2. Use the darkness level slider to adjust the darkness of the webpage.
3. Customize the color scheme, font size, and background pattern using the provided options.
4. Preview the changes in the preview section.
5. Click the "Apply" button to apply the changes to the current webpage.

## Troubleshooting and Support

If you encounter any issues or have any questions, please refer to the following:

- Ensure that you have the latest version of Chrome installed.
- Make sure the extension is enabled in `chrome://extensions/`.
- If the extension is not working on a specific website, try refreshing the page or restarting Chrome.
- For further assistance, please open an issue on the [GitHub repository](https://github.com/githubnext/workspace-blank/issues).
